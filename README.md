# Sistema Complementar de Apoio à Gestão de Vendas em Empresa de Representação Comercial

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Trabalho de Conclusão de Curso`

`SEMESTRE: 8º`

O projeto consiste no desenvolvimento de uma ferramenta de Business Intelligence (BI) para apoiar a gestão comercial de uma empresa de representação de produtos para piscinas. A proposta busca melhorar a organização e o aproveitamento das informações já existentes, que são extraídas principalmente do sistema ERP utilizado pela empresa, reduzindo atividades manuais e proporcionando maior controle sobre vendas, clientes e produtos.

A solução utilizará recursos de BI para consolidar e analisar os dados, apresentando informações de forma visual e intuitiva por meio de dashboards, indicadores e relatórios gerenciais. Com isso, espera-se apoiar a tomada de decisão, aumentar a eficiência dos processos e contribuir para o crescimento sustentável da empresa.
